---

## 💡 About the Project

This is a responsive and professional registration form built using pure HTML, CSS, and JavaScript. The goal is to demonstrate client-side form validation, modern UI/UX, and create a clean and presentable portfolio sample.

---

## 🧩 Technologies Used

- HTML5
- CSS3 (with Flexbox and responsive design)
- JavaScript (Vanilla JS – no frameworks)
- RTL (Right-to-Left) language support with Persian font

---

## 🛠 Features

- Live input validation while typing
- Dynamic error and success messages with animation
- Strong password checker (length, uppercase, lowercase, number, symbol)
- Toggle show/hide password
- Social login buttons (UI only)
